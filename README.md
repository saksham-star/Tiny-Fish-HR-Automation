🚀 TinyFish HR Automation Platform
Multi-Step Web Automation for Modern Recruitment Teams

📌 Project Overview
TinyFish HR Automation Platform is a production-grade SaaS solution designed to automate time-consuming recruitment workflows using intelligent multi-step web automation and API integrations.

The platform eliminates manual HR tasks such as job aggregation, resume screening, and interview scheduling, enabling recruiters to focus on strategic hiring decisions rather than repetitive administrative work.

🎯 Problem Statement
Recruiters spend 25–30 hours per week on manual processes such as:
Posting and collecting jobs from multiple platforms
Downloading and reviewing resumes manually
Matching candidate skills
Coordinating interviews across calendars
Updating ATS systems

This results in:

Reduced productivity

Delayed hiring cycles

Higher operational costs

💡 Solution

TinyFish automates the complete recruitment pipeline through:

Job Aggregation Automation

AI-Based Resume Screening

Automated Interview Scheduling

The system performs multi-platform integration, real-time execution tracking, and workflow orchestration using a scalable backend architecture.

🔧 Technical Stack

Frontend: React (Interactive dashboard & workflow tracking)

Backend: Node.js + Express (REST APIs & automation engine)

Standalone Demo: HTML-based live execution simulation

Integration APIs: ATS systems, job platforms, calendar services

Deployment: Cloud-ready (GitHub Pages, Netlify, Heroku, Railway)

⚙️ Core Features
1️⃣ Job Aggregation Workflow

Navigates multiple job platforms

Handles authentication & pagination

Collects job listings in real time

Displays live execution logs

Impact: Saves ~8 hours per week per recruiter

2️⃣ Resume Screening Workflow

Connects to ATS platforms

Downloads & parses PDF resumes

Applies NLP-based skill matching

Generates candidate scores (0–100 scale)

Impact: Saves ~12 hours per week

3️⃣ Interview Scheduling Workflow

Integrates with Google & Outlook calendars

Identifies mutual availability

Automatically schedules meetings

Sends confirmation emails

Updates ATS records

Impact: Saves ~6 hours per week
